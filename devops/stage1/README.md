# HNG Stage 1 API

A simple REST API built with Node.js and Express.

## Run Locally

```bash
npm install
node index.js
```

## Endpoints

| Endpoint | Method | Response |
|----------|--------|----------|
| `/` | GET | `{"message": "API is running"}` |
| `/health` | GET | `{"message": "healthy"}` |
| `/me` | GET | `{"name": "...", "email": "...", "github": "..."}` |
